Nuestra historia

En el instituto que estudiamos (TECSUP), hemos podido notar que la cantidad de estudiantes disminuye ciclo tras ciclo y no necesariamente es por temas económicos. Además, para tratar de disimular eso, hacen intercambios; sin embargo, una parte de ese grupo (por no decir mayoría) provocan el mismo error del principio. Así que, para evitar este tipo de problema, decidimos crear esta página web.

Herramientas utilizadas

Durante el aprendizaje de nuestra carrera, hemos aprendido a utilizar diferentes herramientas tecnológicas y quicimos representar todo ese conocimimento en esta página web.

Equipo de Trabajo

Este trabajo no se hizo solo, somos 3 estudiantes de Tecsup con habilidades para la programación, cada uno con su respectivo rol de Frontend y Backend.
